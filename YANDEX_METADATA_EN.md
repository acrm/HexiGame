# Yandex Games Metadata (EN)

**Title**: Hexi

**SEO Description** (<=160):
A hex‑grid puzzle where you guide a turtle, capture colored tiles, and build pleasing color patterns.

**About the Game** (<=1000):
Hexi is a minimalist puzzle on a hexagonal grid. Control the turtle, capture colored tiles, and move them to empty cells to form strong spatial patterns. Capture success depends on how close the tile color is to the base orange. It’s a calm, thoughtful game focused on planning and precision.

**How to Play** (<=1000):
The current public release is mobile-only. Tap reachable cells to move; you can drag to preview focus and release to commit movement.
Use the center action button (`SPAWN` / `DROP`) for context action on the focused cell.
Use the 6-slot ring around it for context transfer (`EAT` / `PUT` / `SWAP` hints are shown directly on slots).
On the start screen, choose your language (EN/RU), continue an unfinished session, or start a new one.
