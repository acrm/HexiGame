# Yandex Games Metadata (RU)

**Название**: Hexi

**Описание для SEO** (<=160):
Паззл на гексагональной сетке: перемещай черепашку, захватывай клетки и собирай цветные узоры.

**Об игре** (<=1000):
Hexi — минималистичная головоломка на гексагональной сетке. Вы управляете черепашкой, захватываете цветные клетки и переносите их на пустые позиции, чтобы собирать красивые узоры. Шанс захвата зависит от расстояния цвета до базового оранжевого. Игра рассчитана на спокойное планирование и внимательные решения.

**Как играть** (<=1000):
Сейчас доступна мобильная версия. Тапайте по доступным клеткам; можно вести палец для предпросмотра фокуса и отпустить на нужной клетке для перемещения.
Центральная кнопка действия показывает `SPAWN` или `DROP` и выполняет контекстное действие для фокусной клетки.
Вокруг неё — кольцо из 6 слотов хотбара: при тапе выполняется перенос по контексту (`EAT` / `PUT` / `SWAP` подсказки показываются на слотах).
На старте можно выбрать язык (EN/RU), продолжить незавершённую сессию или начать новую.
